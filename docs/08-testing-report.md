# 8. Тестілеу есебі

## 8.1 Тестілеу стратегиясы

| Тестілеу түрі | Құрал | Ауқымы |
|---------------|-------|--------|
| **Unit тесттер** | Vitest | Утилит функциялар, валидация |
| **Интеграциялық** | Manual + API | API endpoint-тер, DB операциялар |
| **UI тестілеу** | Manual | Барлық беттер, формалар |
| **Қауіпсіздік** | Manual audit | RBAC, XSS, rate limit |
