# 9. Пайдаланушы нұсқаулығы (User Guide)

## 9.1 Жүйеге кіру

### Тіркелу
1. Браузерде `http://localhost:3000/register` ашыңыз
2. Аты-жөніңізді, email-ді, телефон нөмірін және құпиясөзді енгізіңіз
3. Рөлді таңдаңыз: **Пациент** немесе **Дәрігер**
4. «Тіркелу» батырмасын басыңыз
5. Email-ге жіберілген 6-цифрлы OTP кодты енгізіңіз
6. Аккаунт расталады → автоматты кіру

### Кіру
1. `http://localhost:3000/login` ашыңыз
2. Email мекенжайын енгізіңіз → «Жалғастыру»
3. Құпиясөзді енгізіңіз → «Кіру»
4. 2FA қосылған болса → Google Authenticator кодын енгізіңіз

### 2FA қосу
1. Профиль/Баптаулар бетіне кіріңіз
2. «Екі факторлы аутентификация» бөлімін табыңыз
3. «Қосу» батырмасын басыңыз
4. QR-кодты Google Authenticator қосымшасымен сканерлеңіз
5. 6-цифрлы кодты енгізіп растаңыз

## 9.2 Пациент нұсқаулығы

### Дәрігерді іздеу және жазылу
1. Сол жақ менюден «Дәрігерлер» бөлімін ашыңыз
2. Мамандық бойынша сүзгілеңіз (ортодонт, хирург, т.б.)
3. Дәрігерді таңдаңыз
4. Күнтізбеден бос күнді таңдаңыз
5. Бос слотты таңдаңыз
6. Шағымыңызды жазыңыз
7. «Жазылу» батырмасын басыңыз

### Қабылдауларды басқару
1. «Қабылдаулар» бөлімін ашыңыз
2. Жоспарланған қабылдауларды көріңіз
3. Қажет болса «Бас тарту» батырмасын басыңыз

### Профильді өңдеу
1. Сол жақ менюден «Профиль» бөлімін ашыңыз
2. Жоғарғы оң жақтағы **«✏️ Өзгерту»** батырмасын басыңыз
3. Мыналарды толтырыңыз:
   - **Облыс** — Қазақстан облыстарынан таңдаңыз
   - **Туған күні** — күнтізбеден таңдаңыз
   - **Жынысы** — Ұл/Қыз
   - **Қан тобы** — I+, II-, III+, т.б.
   - **Аллергия** — тиісті чекбокстарды белгілеңіз
   - **Мекенжай** — толық мекенжайды жазыңыз
   - **Ескертпе** — қосымша медициналық ақпарат
4. **«💾 Сақтау»** батырмасын басыңыз
5. Өзгертуден бас тартқыңыз келсе → **«Бас тарту»** батырмасын басыңыз

### Емдеу жоспарын көру
1. «Медициналық тарих» бөлімін ашыңыз
2. Емдеу жоспарларын көріңіз
3. PENDING статусты жоспарды «Бекіту» немесе «Қабылдамау»

### Дәрігермен чат
1. «Хабарламалар» бөлімін ашыңыз
2. Дәрігерді таңдаңыз
3. Хабарлама жазыңыз немесе файл жіберіңіз
4. Барлық хабарламалар шифрланған — тек сіз бен дәрігер көре алады

## 9.3 Дәрігер нұсқаулығы

### Дашборд
- Бүгінгі қабылдаулар автоматты көрсетіледі
- Қабылдауды аяқтау: «Аяқтау» батырмасы
- Пациент келмеді: «Келмеді» батырмасы

### Кесте баптау
1. «Профиль» бөлімін ашыңыз → «Өзгерту»
2. Жұмыс күндерін таңдаңыз (Дс, Сс, Ср, Бс, Жм)
3. Жұмыс уақытын орнатыңыз (мыс. 09:00 – 18:00)
4. Слот ұзақтығын таңдаңыз (30/45/60 мин)
5. «Сақтау» батырмасын басыңыз

### Емдеу жоспарын құру
1. «Пациенттер» → пациентті таңдаңыз
2. «Жаңа емдеу жоспары» батырмасын басыңыз
3. Диагнозды, процедураларды және құнды енгізіңіз
4. «Жоспарды жасау» батырмасын басыңыз
5. Пациентке хабарландыру жіберіледі

### Төлем тіркеу
1. Емдеу жоспары бетінде «Төлем қосу» батырмасын басыңыз
2. Соманы енгізіңіз
3. Төлем әдісін таңдаңыз: Қолма-қол / Карта / Аударым
4. «Тіркеу» батырмасын басыңыз

### Қаржылық есеп
1. «Қаржы» бөлімін ашыңыз
2. Күн диапазонын таңдаңыз
3. Төлем әдісі бойынша сүзгілеңіз
4. Жалпы, орташа, максимал сомаларды көріңіз

---

## 9.4 Админ нұсқаулығы

### Дәрігер бекіту
1. «Дәрігерлер» бөлімін ашыңыз
2. PENDING статусты дәрігерлерді көріңіз
3. «Бекіту» немесе «Бас тарту» батырмасын басыңыз

### Қауіпсіздік журналы
1. «Қауіпсіздік» бөлімін ашыңыз
2. Аудит логтарды сүзгілеңіз (оқиға түрі, күн, пайдаланушы)
3. Күдікті әрекеттерді бақылаңыз
