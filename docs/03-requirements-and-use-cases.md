## 3.3 Use Case сценарийлері

### UC-01: Қабылдауға жазылу
- **Актор:** Пациент
- **Ағым:** Дәрігер таңдау → бос слот көру → күн/уақыт таңдау → шағым жазу → жазылу → хабарландыру

### UC-02: Емдеу жоспарын құру
- **Актор:** Дәрігер
- **Ағым:** Пациент таңдау → диагноз жазу → процедуралар қосу → құн енгізу → сақтау → пациент бекіту

### UC-03: Пациент профилін өңдеу
- **Актор:** Пациент
- **Ағым:** «Өзгерту» басу → форма өзгерту → облыс/жынысы/аллергия толтыру → «Сақтау» → аудит лог

### UC-04: Шифрланған чат
- **Актор:** Пациент/Дәрігер
- **Ағым:** Хабарлама жазу → AES-256-CBC шифрлау → сервер сақтау → алушы дешифрлау
