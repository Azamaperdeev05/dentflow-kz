## 1.2 Жобаның мақсаты

Стоматологиялық клиниканың жұмыс процестерін автоматтандыру және цифрландыру мақсатында веб-платформа құру. Платформа пациенттердің онлайн жазылуын, дәрігерлердің кестені басқаруын, емдеу жоспарларын жүргізуін, қаржылық есепті қалыптастыруын және қауіпсіз коммуникация құралдарын қамтамасыз етеді.

## 1.3 Жобаның міндеттері

1. **Аутентификация жүйесін құру** — Email-OTP верификациялы тіркелу, 2FA (TOTP), құпиясөзді қалпына келтіру
2. **Рөлге негізделген қолжетімділік** — Пациент, Дәрігер, Админ рөлдері бойынша бөлінген панельдер
3. **Қабылдауларды басқару** — Күнтізбелі интерфейс арқылы слот сайлау, жазылу, өзгерту
4. **Емдеу жоспарын жүргізу** — Кезеңдік жоспар, диагноз, процедуралар, пациент бекітуі
5. **Қаржылық модуль** — Төлем тіркеу, әдіс бойынша статистика, сүзгілер
6. **Коммуникация** — Шифрланған чат (AES-256-CBC), файл жіберу, хабарландырулар
7. **Қауіпсіздік** — Аудит лог, rate limiting, XSS/CSRF қорғанысы, тәуекел скоринг
8. **Профиль басқару** — Пациент пен дәрігердің өз деректерін өзгерту мүмкіндігі

## 1.4 Жобаның өзектілігі

Қазақстандағы стоматологиялық клиникалардың басым бөлігі әлі де қағаз тіркеу немесе шектеулі бағдарламалық шешімдер қолданады. DentFlow KZ:

- **Уақытты үнемдейді** — пациенттер онлайн жазылып, дәрігерлер кестені автоматты басқарады
- **Деректерді қорғайды** — AES-256-CBC шифрлау, 2FA, аудит лог
- **Қазақ тілінде** — интерфейс толығымен қазақ тілінде
- **Ашық бастапқы кодты** — MIT лицензиясымен бөлісіледі

## 1.5 Күтілетін нәтижелер

1. Толық жұмыс істейтін веб-платформа (MVP)
2. 3 рөлдік панель (пациент, дәрігер, админ)
3. 30+ API endpoint
4. 16 деректер базасы кестесі
5. Қауіпсіздік аудиті мен тестілеу есебі
6. Толық техникалық құжаттама (11 бөлім)

## 1.6 Зерттеу әдістері

- **Талдау** — бар стоматологиялық бағдарламалардың функционалдық салыстырмалы талдауы
- **Жобалау** — UML use-case, ER-диаграмма, компонент диаграммасы
- **Әзірлеу** — Agile/Scrum методологиясы, итерациялық даму
- **Тестілеу** — Unit тесттер (Vitest), интеграциялық тестілеу, қауіпсіздік аудиті
