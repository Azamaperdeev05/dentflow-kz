# 01. Дипломдық жоба паспорты

## 1. Негізгі ақпарат

- Тақырып: Стоматологиялық клиниканың веб-платформасын жобалау және әзірлеу
- Жоба атауы: DentFlow KZ
- Студент: Алтынбек Назгүл
- Топ: СИБ 22-2

## 2. Зерттеу мақсаты

Стоматологиялық клиниканың негізгі бизнес-процестерін (пациент ағыны, қабылдау кестесі, емдеу жоспары, төлем, коммуникация, қауіпсіздік) біртұтас веб-платформада автоматтандыру.

## 3. Зерттеу объектісі және пәні

- Объект: Стоматологиялық клиникадағы ақпараттық процестер.
- Пән: Клиникалық қызметті цифрландыруға арналған веб-бағдарламалық архитектура, деректер құрылымы және API логикасы.

## 4. Міндеттер

1. Пәндік аймақты талдау және талаптарды қалыптастыру.
2. Рөлдік модель құру: пациент, дәрігер, админ.
3. Деректер базасының логикалық/физикалық моделін жобалау.
4. API және бизнес-логиканы іске асыру.
5. Қауіпсіздік механизмдерін енгізу (аутентификация, 2FA, RBAC, аудит).
6. Тестілеу және сапа бағалау.
7. Енгізу және сүйемелдеу нұсқаулығын дайындау.

## 5. Әдістер және құралдар

- Жүйелік талдау, салыстырмалы талдау, модельдеу.
- Full-stack веб-әзірлеу әдістері.
- Технологиялар: Next.js, React, TypeScript, Prisma, SQLite, NextAuth, Zod, Tailwind CSS, Vitest.

## 6. Ғылыми және практикалық маңыз

- Ғылыми маңыз: клиникалық процестерді рөлдік және оқиғаға негізделген архитектура арқылы ұйымдастыру тәсілі.
- Практикалық маңыз: клиника жұмысының ашықтығы, уақытты үнемдеу, қателерді азайту, ақпарат қауіпсіздігін арттыру.

## 7. Жоба нәтижесі

- Жұмыс істейтін веб-платформа (DentFlow KZ).
- REST API және толық деректер моделі.
- Қауіпсіздік аудиті мен тәуекел сигналдарын бақылау модулі.
- Дипломдық жұмысқа жеткілікті техникалық және әдістемелік құжаттама.

## 8. Кілт сөздер

Стоматология, веб-платформа, клиника, Next.js, Prisma, SQLite, API, RBAC, 2FA, аудит, тәуекел, автоматтандыру.
